RAG Q&A Chatbot on Research Papers

Description: Built a Retrieval-Augmented Generation (RAG) chatbot using Gemini 1.5 Pro to answer questions based on academic papers like 

"Leave No Context Behind". Integrated ChromaDB for semantic retrieval and Streamlit for interactive UI.

Tech: LangChain, ChromaDB, Gemini Pro, Streamlit, Google Generative AI API, Python
